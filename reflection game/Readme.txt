
Compilation - 
cd into the folder 'OPEN GL' and run the command 'make'

Execution -
The input file is supplied. It is named 'in'.

To run the code, run the command ' ./name in '

The input file can be changed as per requirement.

To enter/exit the random walk mode, press 'z'
To enter/exit the transport mode, press 't'
To enter/exit the gaze mode, press 'g'

Multiple modes can be run at the same time.
output file is created as 'output'
